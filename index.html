MyCellar

Personal project to improve my Swift skills.

Me and my girlfriend had been facing an issue when we are shopping - sometimes we didn’t know the current stock of our items at home and we bought too many or no need. Also, sometimes we didn’t know if the price of a specific item is a good deal or not. Of this need was born MyCellar, an application to manage the inventory, to create shopping list and to record a price history. The application has some other features like scan barcodes to easily find an article, add stores, record price, share shopping lists and also create categories and items.

  • Date: started Dec 2018

Ricardo.ch

Ricardo.ch is the biggest second-hand marketplace of Switzerland. For 2 years, I had the opportunity to work in 2 separated projects Ricardo.ch and Auto-Ricardo, one in Xamarin framework (iOS & Android) and other in native (iOS & Android), respectively. I’ve worked in a Scrum team of 8 members and it was a big challenge to learn and work with all stack technologies, processes and, at the same time, maintaining 2 applications with a release cycle of every 2 weeks. Our team was also responsible to build our mobile API to access the data, so I had the opportunity to work with Microservices using Go language.

  • Date: January 2017 to January 2019
  • Mobile Developer
  • Xamarin, C#, Go, Swift, Java

ChildSafe

ChildSafe is a platform to monitoring children using a device to integrate in a shoe or using as bracelet. During my master thesis, I decided to study NoSQL databases and how well they solve BigData and, most important, how they adapt with different kind of information. Therefore, ChildSafe was born as proof of concept using CouchDB server, a document DB. I’ve developed a web and mobile applications and given a tiny device with GPS, battery status, temperature and some others features, the platform was able to adapt, tracking the child and notify the caregiver. See the demo in the video.

  • Date: September 2014
  • NoSQL, PHP, Java, WebSockets

ISEP Ticket

In Mobile System, a course in my masters in Computer Science, I had as final work to develop an application in Android. My project proposal was to develop a ticket validation system through QrCodes which the main goal was to allow users to make an access codes’s request of the services that uses queues and restricted access. To do that I’ve developed an application to authenticate QrCodes through camera feature, and a second application which allow users to sign-up or login, request tickets for queues and restricted access. In the case of queues systems, the users receive updates of the number called by push notification. You can see some screenshots in the image above.

  • Date: April 2013
  • Android, Java, ZXing library, Google Cloud Messaging, PHP, MySQL
